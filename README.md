# UAS Sistem Terdistribusi - Darryl & Farhan

Repositori ini berisi implementasi Microservices menggunakan Docker dan Cloudflare Tunnel di STB.

### Struktur
- `/microservices`: Berisi layanan inti Katalog Buku (Port 8002).
- `/integrasi`: Berisi layanan agregator yang menggabungkan data Katalog dan Review (Port 8003).
